# Lost Signal

Lost Signal — минималистичное dark-приложение про уход от цифрового шума.

Это не трекер привычек, не productivity tool и не челлендж.  
Это тихий экран, который показывает, как далеко пользователь ушёл от шума: коротких видео, соцсетей, новостей, игр или другого цифрового фона.

Интерфейс построен вокруг образа ночного северного леса: туман, слабый сигнал, стеклянные панели, приглушённый зелёный акцент и минимум текста.

## Идея

Пользователь выбирает, что для него является шумом, указывает дату начала пути и первый рубеж. После этого Lost Signal показывает:

- сколько дней прошло без выбранного шума;
- сколько времени вернулось;
- где пользователь сейчас на маршруте;
- сколько осталось до первого рубежа;
- когда достигнут Lost Signal.

Главная мысль продукта:

> Уйти от шума.

## Что уже реализовано

- fullscreen onboarding с выбором шума, времени, даты начала и первого рубежа;
- сохранение пути пользователя в `localStorage`;
- главный dashboard с днями, возвращённым временем, текущей локацией и маршрутом;
- тёмный лесной фон, fog/noise layers и спокойные анимации;
- navigation назад внутри onboarding;
- flow `Начать заново` с подтверждением;
- адаптивная вёрстка для desktop и mobile.
