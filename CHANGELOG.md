# Changelog

История версий и изменений проекта Lost Signal.

## v0.1.0 — MVP base

Добавлено:

- базовая MVP-версия главного экрана;
- крупный счетчик дней;
- блок с возвращенными часами;
- текущая локация;
- дата начала;
- базовая шкала прогресса;
- темный лесной визуальный стиль.

Планируется:

- улучшить дизайн интерфейса;
- добавить эффект дождя;
- добавить туман и атмосферные слои;
- улучшить типографику;
- добавить плавные анимации;
- адаптировать интерфейс под мобильные устройства.
